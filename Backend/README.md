# Backend — Next.js API

Next.js backend/API for Tatweer Hackathon.

## Requirements

- Node.js 18+
- npm / yarn / pnpm

## Setup

```bash
npm install
npm run dev
```

Runs on `http://localhost:3000`.

## Structure

```
backend/
├── app/
│   └── api/        # API routes
├── lib/            # Utilities
└── .env.local      # Environment variables (not committed)
```
